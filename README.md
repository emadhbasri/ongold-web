## 📖 About The Project

**Ongold** is a live gold trading platform in Iranian Rial (Toman).

Users can:
- ✅ Register & verify identity (KYC)
- 💰 Deposit & withdraw funds
- 📊 Build and manage investment portfolios
- 🥇 Trade gold ounces (Spot & Pending orders)
- 🛑 Set Stop-Loss (SL) & Take-Profit (TP)
- 🔒 Auto-close portfolio on critical loss threshold
- 📈 Trade with multiple lots
- 👥 Referral system with commission earnings
